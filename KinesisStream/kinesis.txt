Amazon Kinesis cost-effectively processes and analyzes streaming data at any scale as a fully managed service. 
With Kinesis, you can ingest real-time data, such as video, audio, application logs, website clickstreams, 
and IoT telemetry data, for machine learning (ML), analytics, and other applications.
Amazon Kinesis Data Streams is a serverless streaming data service that simplifies the capture, processing, and storage of data streams at any scale.
Amazon Kinesis also assists with processing and analyzing data as it reaches and responds instantly without having to wait for the entire collection of data so that the processing could begin.
